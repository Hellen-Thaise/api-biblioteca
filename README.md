# api-biblioteca
API de biblioteca feita como exercício de fixação do módulo de Spring

## Funcionalidades:
- Exibir todos os livros
- Exibir um livro específico
- Cadastro de um novo livro
- Alteração de um livro
- Deleção de um livro

### Atributos:
- Nome
- Autor
- Data de lançamento
- Código do livro

#### Ferramentas:
- Java 11
- Spring Framework
- Docker
- DBeaver
- Postman
